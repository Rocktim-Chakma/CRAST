# Results

This folder contains the central result tables and audit records supporting the
CRAST paper experiments.

## Contents

| Folder | Purpose |
|---|---|
| `tables/` | Reviewer-friendly central copies of the main paper result tables. |
| `audit/` | Hash manifest, verification files, feature schema, and audit metadata. |

Manuscript figures are stored separately under:

```text
Figures/
```

This `results/` folder intentionally does not contain figure files. Numeric
result values are provided as machine-readable CSV tables under `results/tables/`.
